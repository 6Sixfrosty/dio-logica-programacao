# 🏆 Classificador de Nível de Herói

## 📌 Descrição

Este projeto foi desenvolvido como desafio de lógica de programação.
O objetivo é criar um sistema que classifique o nível de um herói com base na quantidade de XP informada pelo usuário.

## 🎯 Regras de Classificação

- XP menor ou igual a 1000 → Ferro
- XP entre 1001 e 2000 → Bronze
- XP entre 2001 e 5000 → Prata
- XP entre 5001 e 7000 → Ouro
- XP entre 7001 e 8000 → Platina
- XP entre 8001 e 9000 → Ascendente
- XP entre 9001 e 10000 → Imortal
- XP maior ou igual a 10001 → Radiante

## 🚀 Tecnologias Utilizadas

- JavaScript (Node.js)
- Biblioteca prompt-sync

## ▶️ Como Executar

1. Instale as dependências:
   npm install

2. Execute o programa:
   node classificador-nivel-heroi.js
